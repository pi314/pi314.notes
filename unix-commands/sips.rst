===============================================================================
``sips``
===============================================================================
Scriptable Image Processing System

應該是 Mac OS 才有

* 取得圖片大小 ::

    $ sips -g pixelWidth img.png
    $ sips -g pixelHeight img.png

* 調整圖片大小 ::

    $ sips --resampleWidth pixelsW
    $ sips --resampleHeight pixelsH
