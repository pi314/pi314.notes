===============================================================================
`youtube-dl <https://github.com/ytdl-org/youtube-dl>`_
===============================================================================

* 下載 youtube 影片 ::

    $ youtube-dl URL

* 查詢特定 youtube 影片有哪些可用的格式 ::

    $ youtube-dl URL -F

* 指定下載的格式 ::

    $ youtube-dl URL -f <format code>

* 指定高品質音訊 / 指定高品質視訊並合併 ::

    $ youtube-dl URL -f bestvideo+bestaudio
